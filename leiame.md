# Treinamento Neo4j - Análise de Dados com Grafos

Este repositório contém materiais e exercícios relacionados ao treinamento Neo4j - Análise de Dados com Grafos oferecido pela [DIO](https://web.dio.me/track/neo4j-analise-de-dados-com-grafos).

## Descrição

O treinamento Neo4j - Análise de Dados com Grafos visa capacitar profissionais no uso de bancos de dados de grafos para análise de dados, abrangendo desde conceitos básicos de teoria dos grafos até técnicas avançadas de consulta e análise usando o Neo4j. Este bootcamp enfatiza aplicações práticas em áreas como redes sociais, detecção de fraudes e sistemas de recomendação.

## Conteúdo

- Introdução aos Bancos de Dados de Grafos e Neo4j
- Fundamentos da Linguagem de Consulta Cypher
- Modelagem de Dados e Importação de Dados
- Consulta e Análise Avançadas
- Algoritmos de Grafos e Análises
- Projetos Práticos e Desafios
- Melhores Práticas e Implantação



## Pré-requisitos

- Conhecimento básico de programação (ex.: Python, Java ou JavaScript)
- Compreensão básica de bancos de dados e estruturas de dados
- Familiaridade com ferramentas de linha de comando ou IDEs como VS Code
- Proficiência em português (já que os materiais do bootcamp estão em português)
